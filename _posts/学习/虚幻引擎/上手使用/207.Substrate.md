# Substrate 是什么？

- 它是 新的材质框架，在 5.3 里是实验特性（需要在项目设置里开启）。
- 不再只是输出一个传统的 “Base Color + Metallic + Roughness” 的 Surface，而是通过 BSDF（双向散射分布函数） 来描述材质。
- 支持 更真实的物理分层（比如金属 + 清漆层、皮肤表面 + 油脂层、车漆多层等）。

# UE 5.3 怎么开启 Substrate

Edit（编辑） → Project Settings（项目设置）

Enable Substrate

重启编辑器

# 开启 Substrate 对现有材质的影响

开关“开启 Substrate”本身，不会直接破坏你现有的材质。真正有影响的是你是否把某个材质改成使用 Substrate 输出。

当然，最后的节点还是会变化的。

我们可以在最后使用 Substrate UE4 Default Shading 连接 最后节点的 Front Material 这样就没有使用 substrate 的任何特性功能了。
