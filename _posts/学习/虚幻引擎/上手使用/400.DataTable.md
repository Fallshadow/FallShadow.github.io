# 简介

## 是什么？

本质：DataTable 是 UE 中的结构体表格数据容器，一行对应一条记录，一列对应一个结构体字段。

每个 DataTable 必须绑定一个 UScriptStruct

## 做什么？

常见用途：

配表：怪物属性、武器表、关卡配置、任务表、数值表

# 导出

可以直接右键导出为 json 或者 csv

# 导入

直接把 json 拉进来
