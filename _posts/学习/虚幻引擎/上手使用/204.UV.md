# UV

uv 就是 2D 纹理坐标。它将 2D 纹理贴图准确地贴到 3D 模型表面。

一般可以使用乘法节点一起缩放 uv

变量遮罩 和 变量合并  分别联合控制 uv

time 节点 制作材质动画 可以用来流动 uv
