    "api_id": 1000,
    "data": {
        "username": "player_1",
        "version": "v0.7.0.240812",
        "close_heartbeat": true,    // 心跳，测试时候关闭
        "wx_code": "123",
        "avatar_url": "url"
    }
}


消息包体结构

api_id
int
消息ID，用来标记消息，便于解析消息结构


typ
int
消息类型 0请求 1响应 2推送


data
object
消息内容，根据消息id会有不同的消息结构
