# 环境变量配置文件

在 Windows 系统的 Documents 文件夹下的 C:\Users\{your_user_name}\Documents\houdini18.5 中，使用文本编辑器打开名为 houdini.env 的文件。

houdini.env = Houdini 启动时会读取的环境变量配置文件

HOUDINI_PATH = C:/my_tools;D:/CitySampleSource/Small_City/houdini;&

将 my_tools houdini 和 houdini 默认路径下的文件导入进去。也就是用 ; 分割（如果写成两行，后面的会覆盖前面的）。这里注意 & 就是指 houdini 默认路径文件，不可以省略。

要注意这个会导致每次启动的 houdini 都会导入这些。

# 在哪里看得到？

主菜单的 资产（Asset）> 资产管理器（Asset Manager） 下查看。展开 运算符类型库（Operator Type Libraries）> 扫描的资产库目录（Scanned Asset Library Directories） 可查看 .hda（Houdini数字资产）文件列表。
