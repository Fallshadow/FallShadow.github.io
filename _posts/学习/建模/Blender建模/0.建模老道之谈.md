- 倒角让你的模型没那么硬，更可爱
- 批量小物件：比如 3 层书架上的书，可以高矮胖瘦不一致，可以前后位置不一致，可以倾斜放倒，可以镜像，可以部分复制。达到凌乱效果
- 头盔这种圆形小物件：一般你可能会使用圆形，但其实使用方形然后细分球形化之后会更漂亮，球形化之后再修改器细分，应用之后再次球形化
- 把看不见的面删除减轻负担（这里可以选中看得见的面，然后反选，就是看不见的面了）
- 缩放之后 ctrl + a 清空一下缩放
- 如果你有很多东西都需要一样的修改器，你可以先为一个东西弄好修改器，然后 ctrl + L 选择关联修改器
- 物体模式下，ctrl + L 还可以进行替换数据，比如一个坦克一条履带上的 3 个轮子，你细化好了一个轮子，可以使用 ctrl + L 的关联数据，替换另外两个轮子，具体操作是先选中你制作好的数据，然后再选中另外的两个轮子，然后再关联数据，关联材质
- 布线尽量是均匀地四边形
