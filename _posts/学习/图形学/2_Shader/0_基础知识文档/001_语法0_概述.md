# Shader 结构块

一般有 Properties 块，SubShader 块，Pass 块，CGPROGRAM/ENDCG 块，Tags 块，Fallback 块，CustomEditor 块

## SubShader 和 Pass

SubShader 是一整套 “渲染方案”

一个 Shader 可以写多个 SubShader
用来适配不同平台 / 显卡能力
Unity 会从上到下挑一个能用的

一个 SubShader 里可以有多个 Pass，每个 Pass 做一件事，比如：

第一个 Pass：画基础颜色
第二个 Pass：画高光
第三个 Pass：画阴影


## CGPROGRAM

CGPROGRAM 本质就是：开始写 “真正着色器代码”的地方（GPU要执行的代码）

位于 pass 中

在这里书写顶点和像素着色器
