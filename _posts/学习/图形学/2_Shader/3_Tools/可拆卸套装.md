# 时间控制

## 1：零一切换：自定义 0 存在的时间，1 存在的时间，以及两者相互渐变的时间

核心思路就是利用一个正向 smoothstep 和一个反向 smoothstep。

比较简单的想法是，以一个时间节点为开始，Fract 出循环时间段，比如 0 - 5。

然后直接一个反向 smoothstep 定义从 0 - 0.5，意为 从 1 变化到 0 需要 0.5s，然后再接一个正向 smoothstep 定义从 3 - 3.5，意为 从 0 变化到 1 需要 0.5s，那么中间的 0.5 - 3 这 2.5 秒的时间就是 0 的持续时间，而后面的 3.5 - 5 这 1.5 秒的时间 就是 1 的 持续时间。

至于如何接，可以直接相加两者，然后使用 saturate。

## 2：零一切换变种单硬切：0-1 秒是 0，1-2 秒渐变为 1，然后瞬间回到 0-1 秒的 0

虽然可以用第一个方法做到，但是有更加简单的做法

以一个时间节点为开始，Fract 出循环时间段，比如 0 - 2。

然后直接一个 smoothstep 定义从 1 - 2。这样就可以了。

顺带一提，如果需要两盏灯，第一个从 0-1 秒是 0，1-2 秒渐变为 1，第二个从 0-1 秒渐变为 1，1-2 秒是 0，不需要额外伤脑筋，直接 offset 时间错位就可以。

# 渐变

## Gradient + Gradient Sample

其实就是一个可控颜色带，可以自定义从 0 - 1 之间颜色的变化。

搭配上光照就可以制作不同光照角度下的颜色变化。可以用在各种颜色随着进度变化的效果上。
