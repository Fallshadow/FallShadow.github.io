## tex2D 和 tex2Dlod

tex2D 不能在顶点着色器中使用，tex2Dlod 可以

tex2D() 依赖屏幕空间导数（ddx / ddy）来自动选择 mipmap level
而这些导数只在片元着色器（fragment/pixel shader）阶段才可用

而 tex2Dlod() 不一样，它不依赖导数，需要你手动指定 mip level

tex2D()：自动选 mip。一个像素移动一点点，UV 变化大 → 说明纹理被压缩得很厉害 → 用更高 mip（更模糊）。UV 变化小 → 说明纹理比较“铺开” → 用低 mip（更清晰）

“ 距离远 → 用高 mip ”只是结果，不是原因。真正原因是：远处物体 → 投影到屏幕变小 → 一个像素覆盖更大一块纹理 → UV 变化变快（ddx/ddy 变大）→ mip 变高

这点有一个例子说的很明白，一个贴图被强烈倾斜（斜着看），虽然距离不远，但某个方向 UV 变化很快，导致 mip 仍然变得很高，甚至出现各向异性需求

“各向异性需求（anisotropic filtering）”就是在解决一个各个方向模糊程度不一样的问题。

- 普通 mip：像用一个正方形模糊刷子
- 各向异性：像用一个被拉长的刷子（顺着纹理被压缩的方向拉长）

各向异性过滤（Anisotropic Filtering）做的事情是：不再用一个“方形 footprint”（ddx/ddy 近似正方形），而是用一个拉伸的椭圆 footprint

简单说就是：沿着 UV 变化快的方向：多采样、用更高 mip；沿着变化慢的方向：少模糊、保留细节
