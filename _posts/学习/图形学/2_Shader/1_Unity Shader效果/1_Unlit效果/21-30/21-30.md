# 021_LightEffect

```Cpp
float3 L_Point_Dir = _Light021PointPos - pixel.worldPos;
float3 L_Point = normalize(L_Point_Dir);
float distance = length(L_Point_Dir);
float linearFade = 1 - saturate(distance _Light021PointColorAndRange.w);
linearFade = pow(linearFade,_LightPointLinearFadeSoft);
float4 wrapLambert_P = Diffuse(N, L_Point,_WrapLight);
float4 speculary_P = Speculary(N, L_Point, V,_VHMult,_BlinPhong_Exp, _BlinPhong_Scale);
float4 blinnPhong_P = BlinnPhong(N, L_Point, V,_WrapLight, _VHMult,_BlinPhong_Exp,_BlinPhong_Scale);
if (_LightType == 0)
{
    // return frac(_Light021PointColorAndRange.xyzz * _Light021PointColorAndRange.w * _Light021PointPos.xyzz);
    return _Light021PointColorAndRange.xyzz * linearFade * baseMap * wrapLambert_P * 10;
}
else if (_LightType == 1)
{
    // 距离平方衰减！
    float atten = 1.0 / max(distance * distance, 0.01);
    
    return _Light021PointColorAndRange.xyzz * atten * linearFade * baseMap * blinnPhong_P;
}
```

# 022_VirtualLight

特制 shader，营造出一盏灯的照射效果，只影响此 shader 材质本身。

## 用途

- 人物进入山洞，要额外提高人物的亮度
- 做一些点光面积光的效果模拟

## 实现

使用局部坐标计算距离，使用 exp 函数，表达距离近的有显示，距离远的无显示，加上颜色。

## 进阶

相当于自己创建一个灯光了，只影响这些 shader 下的。
