

# 半透明材质

1. 不需要写入深度，把 ZWrite Mode 设置为 off
2. 设置你想要的混合模式，比如 Alpha Blend 

这两个在代码里就位于 pass 上面，声明全局

```HLSL
Blend SrcAlpha OneMinusSrcAlpha // CPU 告诉 GPU 要用 Blend
ZWrite Off // 关闭深度写入
ColorMask RGBA // 声明四通道
```
