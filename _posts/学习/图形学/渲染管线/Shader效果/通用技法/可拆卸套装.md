# 01 切换，自定义 0 存在的时间，1 存在的时间，以及两者渐变的时间

核心思路就是利用一个正向 smoothstep 和一个反向 smoothstep。

比较简单的想法是，以一个时间节点为开始，Fract 出循环时间段，比如 0 - 5。

然后直接一个反向 smoothstep 定义从 0 - 0.5，意为 从 1 变化到 0 需要 0.5s，然后再接一个正向 smoothstep 定义从 3 - 3.5，意为 从 0 变化到 1 需要 0.5s，那么中间的 0.5 - 3 这 2.5 秒的时间就是 0 的持续时间，而后面的 3.5 - 5 这 1.5 秒的时间 就是 1 的 持续时间。

至于如何接，可以直接相加两者，然后使用 saturate。
