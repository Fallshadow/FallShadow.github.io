# 光栅化在干嘛

光栅化的作用，是把经过变换后投影到屏幕空间的三角形，离散成对应的片元（找到其覆盖的像素），并插值出每个片元的属性。

顶点经过 MVP 后进入裁剪空间进行裁剪，再经过透视除法（必须先裁剪，再透视除法，透视除法就是从四维坐标变成三维坐标）进入 NDC（规范化设备坐标，这个就是 -1 -1 1 1 的盒子），随后视口变换（把 NDC 这个标准范围，映射到实际屏幕像素范围 -1 1 到 0 屏幕分辨率）到屏幕空间。

对于重叠的三角形，它们通常都会产生片元；这些片元可能还会执行片元着色器，除非 Early-Z 等优化提前把被遮挡的片元剔除。

# 隔行扫描

以前的显示设备要成像，都是在屏幕上画很多线，画满整个屏幕就形成了一帧画面

隔行扫描就是说

在第一帧只画1、3、5等奇数线

在第二帧只画2、4、6等偶数线

利用人眼的视觉残留特性，这样人们即发现不了画面的异常，还能使机器工作量减半

如今还有某些视频压缩技术采用了这个思想

（但是隔行扫描会造成严重的 行间错位/梳状伪影，特别是对高速运动的画面来说）

给定一个连续的函数 f(x)，当 x 等于 1 时得到的 f(1)就是 1 的采样

所以采样就是把一个函数离散化的过程

我们要采样的东西就是：给定一个三角形，在像素的中心进行采样，来判断中心是否落在三角形内，逐像素遍历，判断该像素中心是否在三角形内部，输出到屏幕显示

# 采样产生的问题

- 走样：空间采样率不足造成的锯齿
- 摩尔纹：高频细节被错误地看成了低频图案。细密重复图案和采样网格不匹配时，产生的“假性大花纹”。至于原本应该全是横纹却大概率出现曲率，是因为这个东西本身不平整有些有规律的细微变化，导致出现圈。
- 车轮效应：时间采样不足造成的慢转或者倒退感。

原因就是 信号的变化太快了，采样的速度太慢或者精度太低，以至于显示给人的两张画面之间少了很多细节。

# 滤波

滤波就是抹掉一些特定的频率，对应的信号如何发生变化，傅里叶变换可以把一个函数从时域变到频域

右边的图像就是左边的照片通过傅里叶变换得到的，右边图像表示的就是有多少信息，中间部分是低频信息，越往外越高频

如何看懂右侧滤波图
- 位置
  - 中心附近 = 低频
  - 离中心越远 = 高频

- 颜色
  - 白 / 亮 = 这个频率成分很强
  - 黑 / 暗 = 这个频率成分很弱，甚至几乎没有

![alt text](GAMES101_06_Rasterization_图像处理滤波.png)

十字线意味着图片边界循环突变频率，一般的图像边界循环时都会表现地突变，所以会有白色的十字线。

## 高通滤波

在频域空间内完全抹掉低频信号(也就是压黑中间部分)，将结果还原成图像，形成左图。高频的东西在图像上表示的就是图像的边界

为什么高频信息代表着边界？

当某一图像的周围突然发生发生了变化，我们就认为他是边界。比如图中人物的衣服和背景就是由黑色突然变成了灰色。相当于颜色信号突然从黑色变成了灰色，就是出现了高频的变化，即边界。

## 应用

看着这些效果，自然可以想到这个滤波可以应用在 mipmap 生成、轮廓效果、素描效果、锐化效果等。

| 操作/概念 | 频域里怎么做 | 图像上会发生什么 | 典型应用 |  
|---|---|---|---|  
| **保留低频（低通滤波）** | 保留中心，压制外圈 | 图像变模糊，细节减少，边缘变软 | 去噪、平滑、降采样前预滤波 |  
| **保留高频（高通滤波）** | 压制中心，保留外圈 | 大轮廓和缓慢变化被去掉，只剩边缘、纹理、细节 | 边缘检测、细节提取、轮廓增强 |  
| **增强高频** | 外圈频率乘更大权重 | 图像更锐利，但噪声也可能被放大 | 图像锐化、清晰化、增强纹理 |  
| **抑制高频** | 外圈频率减弱 | 锯齿、细噪声、细纹理减少，但会变糊 | 去噪、抗锯齿、压缩前处理 |  
| **带通滤波** | 只保留某一圈频率 | 只留下某一尺度的结构 | 纹理分析、特征提取、特定尺度目标检测 |  
| **带阻滤波** | 去掉某一圈频率 | 去除某一尺度的重复纹理 | 去周期噪声、抑制特定纹理干扰 |  
| **方向滤波** | 只保留某些方向的频率 | 强化或抑制某个方向的边缘/纹理 | 检测横线、竖线、斜纹，去条纹干扰 |  
| **陷波滤波（notch filter）** | 精确挖掉频谱中几个亮点 | 去除周期性条纹、网纹、扫描干扰 | 去摩尔纹、去电源纹波、去周期噪声 |  
| **只看幅度谱** | 看频率强弱，不看相位 | 只能知道“有哪些频率”，不知道结构在哪 | 频率分析、纹理统计、图像特征观察 |  
| **相位信息** | 记录各频率如何对齐 | 决定图像的空间结构和位置 | 图像重建、配准、识别 |  
| **频域压缩** | 丢掉弱的高频或不重要频率 | 视觉上变化不大，但数据量下降 | JPEG 类压缩思想 |  
| **频域去噪** | 去除噪声集中的频带/方向 | 减少随机噪声或周期噪声 | 医学图像、遥感图像、老照片修复 |  

# 卷积

移动窗口（Filter），将窗口中三个数和覆盖信号的三个数做点乘，填到结果中

![alt text](GAMES101_06_Rasterization_移动窗口.png)

![alt text](GAMES101_06_Rasterization_移动窗口2.png)

其实就是信号在任意一个地方，在他的周围做了个平均操作

## 卷积的一些定理

时域的卷积 == 频域的乘积

可以拿到一幅图直接用一个卷积滤波器进行卷积操作

也可以

1. 先傅里叶变换这幅图，将这幅图变到频域
2. 将卷积滤波器变到频域上
3. 将两者相乘，乘完后得到的频域的结果，将其逆傅里叶变换，变到域上

![alt text](GAMES101_06_Rasterization_时域卷积和频域的乘积.png)

注意了，这里的卷积图是指 1/9 的 3 * 3 格子。

## 卷积图大小和结果的频率关系

比如咱们用一个超级大的box, 用一个比这个比这个图像还要大的这个box, 那得出来结果, 那各处基本都一样, 因为因为把这个所有的这个图像上的这个像素都给加起来了

或者说我们再想一个极端, 如果我用一个超级小的box, 如果这个这个box比一个像素还小, 那不等于是这个完全没有做滤波吗, 对吧, 那也就意味着说, 如果用一个超级小的这个box, 那么它对应的频率上面的范围就会超级大, 他把所有的频率都留了下来.

所以我用一个大的这个box, 我就只能留下更低的频率, 也就是说让, 结果变得更模糊。

左边一列是时域，右边一列是频域。

对a图像进行c的采样，得到e。对应的操作在频域中就是bdf

时域的采样在频域中就就体现为频域信号的复制

在时域里对连续信号做采样，本质上是“拿一个冲激串去乘原信号”；对应到频域里，就会变成“原频谱被复制很多份”。

为什么会产生走样呢？

采样不同的间隔，会引起频谱不同间隔进行复制，所相交的部分就是走样
