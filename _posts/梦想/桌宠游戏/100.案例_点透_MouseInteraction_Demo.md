# 关卡蓝图

## 初步设置

### 1. 拿到设置

Get Game User Settings 是一个全局的蓝图函数，专门用来在运行时读取和修改玩家的游戏设置。

返回一个 UGameUserSettings 类型的对象。这个对象是全局唯一的，里面存着当前玩家所有的游戏选项，比如分辨率、全屏模式、帧率上限、图形质量、音量等等。

### 2. 改变窗口显示模式

Set Fullscreen Mode 是 UGameUserSettings 提供的一个函数，专门用来强制改变游戏窗口的显示模式

- Fullscreen	传统的全屏独占模式
- Windowed Fullscreen	无边框窗口化全屏
- Windowed	标准窗口模式

### 3. 立即应用窗口更改

Apply Resolution Settings 是 UGameUserSettings 里的一个函数，它的作用是把所有待处理的分辨率和全屏模式修改真正地、立即地应用到游戏窗口上

Set Fullscreen Mode 这类函数，只是把你想要的结果先记录在内存里。只有当你调用了 Apply Resolution Settings，引擎才会去执行底层的窗口创建或调整，把所有记录的设置一次性兑现。

- Apply Resolution Settings	轻量级。只应用屏幕分辨率、全屏模式、窗口位置等与画面输出相关的设置。
- Apply Settings	重量级。应用所有设置，包括画质、帧率、分辨率、阴影质量等等，范围广得多。

### 4. 拿到玩家控制，显示鼠标

Get Player Controller：获取当前关卡里控制玩家的那个“大脑”。

Set Show Mouse Cursor：这个函数是 Player Controller 的成员函数，专门用来控制鼠标指针的显示与隐藏。

### 5. CMD 关闭抗锯齿、150 缩放、锁 60 帧

Execute Console Command 是一个蓝图函数，作用是在蓝图里执行你用键盘在控制台（~键）里输入的任意命令。

r.AntiAliasingMethod 0 ：关闭抗锯齿
r.ScreenPercentage 150 ：以 150% 分辨率渲染，然后再缩回 100% 保证渲染正确
t.maxfps 60 ：锁 60
