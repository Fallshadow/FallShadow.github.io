# 打包基础透明方块

1. 遮罩后处理
2. 方块 + 灯光 + 相机
   1. 方块 Render CustomDepth Pass 开启之后，UE 会在正常渲染完场景后，再额外跑一遍，把所有勾选了 "渲染自定义深度" 的物体画到一张看不见的特殊纹理上。这张纹理记录的不是颜色，而是每个像素离相机的距离。场景中没有勾选该选项的物体，在这张纹理上对应的像素就是空值（也就是你材质里用的那个 10000.0）。项目设置：Project Settings → Engine → Rendering → Postprocessing → Custom Depth-Stencil Pass，选择 "Enabled with Stencil"。通常推荐在项目设置里全局开启，以确保任何时候都有这个缓冲区的数据可用。
   2. 相机 手动聚焦于桌宠所在的距离，可以用吸管吸到目标物体
