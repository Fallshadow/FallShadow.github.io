# 接口协议与带宽：速度的决定性门槛

- USB 3.2 Gen 2 (10Gbps) ：约 1000-1050 MB/s 这是目前最主流的高速档位，足以跑满入门级 NVMe 或高端 SATA 盘的速度。
- USB 3.2 Gen 2x2 (20Gbps) ：约 2000 MB/s 这是中高端 PSSD 常见的选择，性价比高，但需要电脑接口支持，比较挑设备。
- USB4 / Thunderbolt 3/4 (40Gbps) ：可超 3000 MB/s。这是旗舰级 PSSD 的接口，价格昂贵，通常只有高端笔记本或创意 PC 会配备，主要用于视频剪辑等专业领域。

# 打游戏方面

USB 协议慢：USB 移动固态硬盘，走的是 USB 协议，需要经过主板的 USB 控制器转换信号。这个转换会带来微小的协议延迟，且会轻微增加 CPU 占用。

随机读取差：在游戏这种高负载场景下，USB 的 4K 随机读取 远低于理论值，受 USB 协议严重拖累，不如 SATA 的稳定在 30-50 MB/s

缓外写入持久力差：断崖式下跌更严重 （密封散热差）

USB 供电不稳：机箱前置 USB 口或老接口供电不足，会让移动盘间歇性断开，游戏直接崩溃。

使用环境不稳：移动硬盘挂在外面，一个误碰、线材拉扯，就可能造成游戏中断、存档损坏甚至硬盘故障。
