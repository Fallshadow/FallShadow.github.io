
# 均热板

均热板 (Vapor Chamber)	一个内部真空并填充了少量液体的扁平腔体。核心热量让液体蒸发，蒸汽会充满整个腔体，把热量二维扩散到整个板面，再均匀地传递给焊接其上的热管。	

散热效率极高，能将热量摊得很均匀，尤其适合发热集中的小核心，设计紧凑。	

成本最高，工艺复杂，多见于高端显卡

# 铜底

铜底 (Copper Base)	在核心和热管之间加一块实心的铜块。热量先从核心传到铜块，再由铜块传给焊接在它上面的热管。	

物理强度高、耐用、可靠，热管不会因直接接触而变形，是一种稳妥的“中间人”方案。	

多了层接触面，理论上热阻稍大，效率不如均热板和理想状态的直触

# 热管直触 

热管直触 (HDT)	把原本圆形的热管压扁并打磨平滑，让它们直接接触 GPU 核心，CPU 散热器常用。	

理论上热阻最小，因为核心热量能最直接地传给热管。	

对工艺要求极高。如果热管打磨不平或受压变形，就会产生缝隙，导致部分热管“围观”，效率反而会大打折扣

# 对比

均热板 >> 理想的铜底 > 理想的热管直触 > 有缺陷的热管直触

# 热管

理论传热能力：根据经典的实测数据，单根直径 8mm 的热管理论上能解决约 60W 的热量，而单根 6mm 热管只能解决约 45W。

3x8mm 组合：3 根 × 60W = 180W 的理论解热能力。

4x6mm 组合：4 根 × 45W = 180W 的理论解热能力。
虽然总理论值一样，但现实世界里 8mm 的物理优势非常大。

物理结构碾压：用圆的面积公式来算更直观。6mm 热管的横截面积是 9π，而 8mm 热管的横截面积是 16π。虽然直径只多了 2 毫米，但内部的蒸发空间（导热截面积）却大幅增加了约 44%。这意味着 8mm 热管内部的导热液体更多，能以高得多的效率搬运热量。

效率瓶颈不同：

8mm 热管：单根能力强，属于“精兵作战”，搬运同样的热量所需的热管更少，内部液体回流更顺畅。

6mm 热管：由于管径细，单根最大热传量小，必须堆数量来弥补。但如果 4 根设计不当（比如并排太挤），反而会导致部分热管接触不良或“围观”，实际效能大打折扣。

热管像一根根独立的水管，把热量从核心搬到远处的散热鳍片上。但水管和核心的接触面积有限，容易有“热点”，而均热板（Vapor Chamber）则像一个超大面积的扁平真空散热腔，直接盖在整个核心上

对比热管散热的时候要考虑到 gpu 核心的面积 和 热管本身的粗细，简单来说，如果你的热管很多但是有很多超出了核心的面积，那就相当于浪费，可能在低能耗场景下还不如更少热管的 显卡

# 散热流程

无论采用直触、铜底还是均热板，它们都只是“第一步”——把核心热量高效地“接过来”。热量接下来就必须交给“第二步”，也就是一根根热管。

这些热管内部有毛细结构和液体，受热后液体蒸发，把热量快速传递到远处密集的散热鳍片上。

所以，显卡完整的散热路径是：

GPU 核心 → 均热板/铜底/直触 → 热管（比如你说的 3*8mm） → 散热鳍片 → 风扇吹走热量

# 风扇

某品牌显卡在介绍散热升级时提到，其搭载的 100mm 风扇，与传统的 90mm 风扇相比，风量增加了 40%以上，风压提高了 55%。

你可以把这组数据套进我们的场景里算一笔账：

2 个 100mm 风扇：每个风扇都比 90mm 的强 40%以上，那 2 个加起来的总效能大约是 2 × 1.4 = 2.8，接近 3 个 90mm 风扇的基准效能。

3 个 90mm 风扇：它的总效能可以看作是 3 × 1.0 = 3。

所以你看，即便是“2 个大的”对上“3 个小的”，总散热能力也基本打平。但 2 个大风扇还有两个额外的好处：

噪音更低：大尺寸风扇不需要转得像小风扇那么快，就能吹出同样多的风。这意味着在相同的散热需求下，2 个 100mm 风扇通常会更安静。

效率更高：就如你问的热管一样，风扇也有“质”与“量”的权衡。两个高效率大风扇的组合，往往比三个小风扇的组合更均衡。

当然，最终效果也会受风扇本身的设计（比如扇叶形状、轴承类型）和显卡整体散热结构影响。但单从“3 个 90mm vs 2 个 100mm”这个规格上来看，选那两个更大的，几乎是不会错的选择
