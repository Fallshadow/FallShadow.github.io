# 公里数

一年不到 1 万公里，就看纯油车。纯油车寿命长，你每年也不开太多，所以开个 5、6 年还跟算是个年轻小伙子。

一年要是能开个 2 万公里，那就买电车。

# 会很费钱

车位费，停车费，油费，洗车费，保险费，保养费，车上零件费。

# 挑选

油车看销量。电车，看技术和国标。

## 油车看销量

首先油车寿命长，保值率高。

这时候，销量高就意味着，这种车更易于维修保养更换零件，而且使用个 5-8 年也还能卖个好价钱。

## 电车看技术和国标

电车寿命短，保值率低，一年就能亏个 40%，保险还贼贵，它就是个祸害钱的东西。所以电车就是给有钱人玩的，或者说非常经常用车想享受的人玩的。

就看你有没有今年应该有的技术！

比如 2026 年，10 W 元买电车，你是否有
- 800V 超快充高压平台架构
- 5C 的充电峰值倍率
- 单电机马力 200 匹以上（240KW）

看你的硬件达没达到国标
- 电池：GB 38031-2025
- 出现了危急时刻，出现了热失控的时候，能不能控制住，热失控两小时内不自燃不起火不爆炸
- 门把手是否有机械开关和电子开关两种

# 价格

落地价 = 裸车价 + 购置税 + 车船税 + 交强险 + 商业险 + 骗人费（金融服务费 + 上牌服务费 + 今天喝杯咖啡费 + 提车出门摆手费）

一般来说 落地价 是 裸车价的 1.15 倍

去某车之家，某车帝上看看最近这个车的落地成交价

注意 同一种车，不同地方，价格会有所不同（受运费、地方经销商数量等影响），一般来说大城市买车回比小城市便宜。查车=价也要看 IP

注意，4s 店定金，是大定还是小定。
- 大定：指定金，用户明确要购买该车，一般来说金额较大，如果消费者取消，不可以退还。如果 4s 店违约了，比如车况、配置不符、交车时间太长，都是可以要求赔偿。
- 小定：指订金，金额较小，可以退

付款的时候，一定不要是跟个人，一定是有盖章，有和 4s 店正规本身一起的

交付定金这种大协议之前就要谈好赠品！提出我买个车么脚垫总是要有的吧，行车记录仪，保养项目通通谈好。谈好的赠品也要写到合同里面去。如果只能口头承诺，赠品写不到APP合同里面的，可以录音录像，聊天截图，万一对方返回可以维权三倍赔偿。

如果贷款买车，一定搞清楚利息和手续费。还有提前还款也要罚钱的。贷款买车有没有返点？（现金优惠）。贷款买车，车子是会被抵押的，还完贷款，车子如何解除抵押，有没有手续费。迟了一天还款，有没有宽限期？影不影响征信？

保险，10万元的油车第一年的商业险一般在 3500 - 4500 之间。20万的油车，大概在 5000 - 6000 的样子。电车要比同价位油车贵个 20% 左右。保险也跟地区有关呢，有些地区保险贵。
