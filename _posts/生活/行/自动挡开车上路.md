# 后视镜调节

左边后视镜角度调节：找到左边后视镜的车前门把手，将左边后视的车前门把手调整到左边后视镜的右下角位置。这样能兼顾到近处天空和地面各占一半，而且呢车身占后视的1/4，转弯变道也能够明确咱们的车身位置，也能够避免刮蹭。

右边后视镜角度：在右边后视镜中找到右边后视的车前门把手，将车前门把手调整到右边后视的左下角位置。天空占后剩的1/3，地面占后剩的2/3，而且呢咱们的车身占后世的1/4。转弯变道也能够明确咱们右侧车身位置，也能够避免咱们的右侧车身发生刮蹭

# 启动

起步步骤：

1、踩住刹车；

2、启动按钮；

3、挂D档；

4、释放手刹；

5、观察仪表盘，看有没有红色的指示；

6、慢慢松开脚刹，车子启动！

# 停车

平路停车步骤：

1、踩住刹车；

2、车停稳后挂P档；

3、拉起手刹，熄火。

坡道停车步骤：

1、踩住刹车；

2、车停稳后挂N档；

3、拉起手刹；

4、松开脚刹；

5、挂P档；

6、熄火。

目的在于让车子在斜坡上的作用力释放在手刹上，而不是P档上。如果直接挂P档，下次挂挡开车就会比较吃力。

# 档位

P 停车档，R 倒档。

N 空档，就是你等红绿灯的时候有个空档的可以停下来，稍微等待一下

注意：如果临时停车或等红绿灯时使用P档，可能会造成后方车追尾我车时，对我车变速箱造成损坏。

D 前进档这边有个+，有个-的，它可以切换到手动模式

S档：运动档，发动机转速会升高，动力会更充沛，可在超车时使用

L档：低速挡，扭矩会升高，可在上下坡时使用，爬坡有力，下坡又不会特别快，可不必频繁的踩刹车来制动

# 变道条件

发现后方车辆出现在后视镜的内侧一半，说明此时后方车辆离我们比较远，说明了此时是安全区；

是可以进行变道以及转弯的，那如果说当我们在开车的过程中，后方车辆出现在我们后视镜内侧，一半和外侧一半的中间时，说明呢也是安全区；

但是呢咱们变道时一定要比后方车辆车速快，否则是很有可能发生刮蹭的，咱们变道时一定要谨慎。

那如果说当我们开车时，发现后方车辆出现在我们后视镜的外侧一半呢，此时后方车辆离我们比较近，说明是危险区，不能够变道转弯，变道转弯是很有可能发生刮蹭的。

所以说这就是通过咱们后视镜，来判断与后方车辆的车距，把握一个变道打轮的一个时机。

# 转弯打轮

盯住咱们汽车A柱夹角，只要A柱夹角刚好是重合于前方（车辆底盘底部，或者墙，或者马路牙子）时，此时咱们的车头距离前方的车辆，大概也是2米五左右，这个时候无论将方向盘向左或者向右打满，同样的在转弯时，咱们的车头脚也不会刮蹭到前方的车辆，也能够做到轻松的转过来，这就是把握一个转弯打轮的时机，避免咱们车头发生刮蹭的一个技巧

# 一把出库

需要盯住咱们车辆的这个B柱，随后将咱们的车辆先直直的往前开，再往前开的过程中，你记住，当发现咱们车辆的这个B柱，刚好是越过旁车的车头或者对齐旁侧的车头，此时再将我们的方向盘向旁侧的方向打满方向，往前开，在往前开的过程中，咱们的车身是不会刮蹭到旁边的车辆的，也能够做到轻松一把的出库

所以说出库时咱们就记住看咱们车辆的B柱，只要咱们车辆的B柱过了旁边的车辆，车头或者越过旁边的障碍物，出库时，都能够避免咱们的车身与旁边车辆发生刮蹭

# 倒车入库

倒车入库时，咱们左倒车库就盯住咱们左边后视镜，右倒车库就盯住咱们右边后视镜。

你比如说此时是左倒车入库，咱们就盯住咱们左侧后视镜，只要咱们的左侧后视镜，刚好是越过第二条横线时，也就是刚好对齐这条横线时。

![alt text](左倒车入库.png)

此时将方向盘向它的反方向打满，也就是向右打满，往前开。

再往前开的过程中，盯住咱们的左侧后视镜，只要在左侧后视镜中刚好能够看到左边库角时，也就是越过的第二条横线时，此时停车，随后挂入倒档方向盘，一把向他的方向打满方向，也就是向左打满。

![alt text](左倒车入库停车位.png)

往后倒车，倒车时观察左右两边后视镜，并同时修正咱们车身行驶轨迹，当发现咱们的车位慢慢的进了侧线，并且左右两个车身平行于侧线，此时回正方向往后倒车，这样呢咱们的倒车入库就能够做到，轻松一把入库

# 侧方位停车

侧方位停车，特别是前后都两边都有车，很多的朋友遇到这样的空车位，不知道该怎么停，其实你只要记住，首先发现目标后，咱们只需要将我们的车辆先直直的往前开，顺着这个车位线保持50公分，直直的往前开，咱们可以盯住雨刮鞋的轨迹点，让雨刮器的股极点，一直扫着前方的这条库边线行驶

![alt text](雨刮器的鼓节点扫标线.png)

当我们在行驶的过程中，这个时候盯住咱们车辆的右侧后视镜，你只要发现，当在咱们的右侧后视镜中，看到咱们与旁车的后视镜对齐时，此时停车，随后挂入咱们车辆的倒档，将方向盘一把向库的方向打满方向，往后倒车

![alt text](和前车位平齐.png)

倒车的同时观察咱们的左侧后视镜，在咱们的左侧后视镜中，只要刚好看到后方车辆完整车头时，此时咱们就可以回正方向往后倒车

![alt text](打满倒车点位.png)

倒车的过程中，这个时候盯住咱们的右侧后视镜，只要在右侧后视镜中，看到咱们马路牙子即将在后视镜中消失时，此时停车

![alt text](回正倒车点位.png)

随后再将方向盘一把向左打满，往后倒车，这样呢咱们的车辆就能够做到轻松一把入库，看到咱们的车身平行侧线，回正方向停车即可
